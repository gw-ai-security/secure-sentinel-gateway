# Role Skill: AI Program and Governance Lead

## Mission of the role
Provide program-level governance, phase oversight, and stakeholder alignment to ensure the AI gateway delivers compliance, security, and documentation outcomes.

## Responsibilities
- Define program governance cadence and decision gates.
- Coordinate cross-role dependencies and escalations.
- Ensure adherence to compliance, documentation, and Knowledge Base rules.
- Align artifacts with stakeholder expectations and portfolio goals.

## Knowledge Base interaction (read/write duties)
- Read: `docs/knowledge_base/architecture_decisions.md`, `docs/knowledge_base/lessons_learned.md`.
- Write: Record governance decisions and phase approvals in `docs/knowledge_base/architecture_decisions.md`.
- Verify: Each phase updates at least one Knowledge Base file with clear ownership.

## Inputs and Outputs (artifacts)
Inputs:
- Phase deliverables from all roles.
- Compliance and risk notes.
- Traceability and quality checks.
Outputs:
- Governance decision records.
- Phase approval notes.
- Risk and escalation summaries.

## Definition of Done
- Governance cadence is documented and followed.
- Phase approvals are recorded and traceable.
- Cross-role dependencies are resolved or escalated.
- Knowledge Base updates are validated for each phase.

## Typical Failure Modes
- Missing or informal phase approvals.
- Untracked decisions and escalations.
- Knowledge Base updates not enforced.
- Misalignment between artifacts and stakeholder needs.

## Handoff rules to other roles
- Provide phase approval notes with decisions and open risks.
- Require Knowledge Base update confirmation at handoff.
- Escalate unresolved issues with clear ownership.

## Work Reporting (Mandatory)
- After completing any work package or phase task, update:
  docs/work_reports/roles/ai_program_governance_lead/phase_<N>.md
- Add a new entry using the official template from docs/work_reports/README.md
- Link to:
  - all created/updated artifacts (files)
  - any Knowledge Base updates (docs/knowledge_base/...)
- The task is NOT considered done until this report entry exists.
